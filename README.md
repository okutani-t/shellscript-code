# shellscript-code
シェルスクリプトのコード

## USAGE

### findgrep

$ ./findgrep.sh 検索文字 パス ファイル名

```
$ ./findgrep.sh hoge
$ ./findgrep.sh hoge /to/path/
$ ./findgrep.sh hoge /to/path/ test.text
```

### replace_filenames

$ ./replace_filenames.sh 変更前の文字列 変更後の文字列 ファイル名

```
$ ./replace_filenames.sh JPG jpg hoge*
```
